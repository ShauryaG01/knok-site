# knok-site
Static SEO site for knok.work. Produced by the knok content factory (Oracle VPS);
served by GitHub Pages; pages drip-published throughout the day.
- `output/` — the live site (only drip-published pages + root)
- `.github/workflows/deploy.yml` — Pages deploy on push
- `.github/workflows/indexing.yml` — IndexNow + Google Indexing API drip
